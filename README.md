## 📖 Descripción

**BrainyKids** es una plataforma educativa interactiva diseñada para niños de **3 a 6 años**, incluyendo aquellos con **TDAH**.

Permite aprender:
- 🔢 Números  
- 🔤 Letras  
- 🎨 Colores  
- 🔷 Figuras  

La API gestiona:
- 👶 Usuarios  
- 📚 Módulos educativos  
- 📈 Progreso  
- 🏆 Recompensas  
- 🔐 Autenticación con JWT
